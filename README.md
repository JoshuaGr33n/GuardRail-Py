## 🎯 Project Vision

GuardRail-Py automates code review checks I routinely perform as a **Lead Engineer** mentoring junior developers.
Rather than relying on regex-based scanning, it leverages Python’s **Abstract Syntax Tree (AST)** to understand code structure, enabling more accurate detection of:

* 🔒 **Security vulnerabilities** (hardcoded secrets, dangerous functions)
* ⚡ **Performance issues** (O(n²) algorithms, inefficient patterns)
* 📚 **Maintainability concerns** (missing documentation, style violations)
